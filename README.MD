## ⚔️ Automated Wave-Based Mob Arena
Technology: Minecraft Education + MakeCode

## Technical Description
A progressive wave system built using state management and global variables. The project simulates a professional game loop where difficulty scales automatically based on player performance.

## The system features:

Wave Counter: Tracks current progress and triggers difficulty spikes.
Scoring System: Dynamic point allocation based on mob types.
Life/Health Controller: Manages player respawns and session limits.
Progressive Spawning: Incremental scaling of entity count and strength.
Win/Loss Conditions: Defined logic for session termination and victory triggers.

## Core Concepts
Game Loop Architecture: The continuous cycle of checking game states.
Global State Control: Managing variables accessible across all functions.
Progression Logic: Designing mathematical curves for difficulty.
Event-Driven Execution: Triggering actions based on in-game occurrences (e.g., "on mob died").

## Developed Competencies
Dynamic Systems Modeling: Creating environments that react to player input.
Complex Flow Control: Managing multiple simultaneous logical branches.
Strategic Progression Design: Balancing game mechanics for engagement.
Critical Decision Making: Analyzing system data to optimize code performance.

## 📥 How to Run (Instructions for Students)
1. Import the World
MEE: Click Import on the home screen and select the .mcworld file.
Bedrock: Double-click the file to launch Minecraft and auto-import.
Java: Convert the folder using a tool like Chunker and place it in your saves directory.

2. Initialize the Arena
Once in the world, look for the Start Button or type the command specified in the code (usually run or start).
Press 'C' to open the MakeCode editor to inspect the Wave Logic and Global Variables.

3. Modifying the Challenge
Easy Mode: Locate the "Spawn Count" variable and decrease the value.
Hard Mode: Increase the "Wave Multiplier" to spawn more mobs per round.

## 📥 Como Importar e Jogar (Passo a Passo)
Como este mundo foi exportado do Minecraft Education Edition (MEE), siga estas instruções de acordo com a sua versão:

## 🎮 Para Minecraft Education Edition (MEE)
Abra o Minecraft Education.
Na tela principal, clique em Importar.
Selecione o arquivo .mcworld que você baixou.
Após a importação, encontre o mundo em "Meus Mundos" e clique em Jogar.

## 💻 Para Minecraft: Java Edition
Observação: o Java Edition não oferece suporte nativo a arquivos .mcworld. Você precisará de uma ferramenta de conversão (como o "Chunker.app") para convertê-lo em uma pasta compatível com Java.

Converta o arquivo para uma pasta Java usando um conversor online.
Pressione Win + R, digite %appdata% e acesse .minecraft/saves.
Cole a pasta convertida em saves.
Abra o Minecraft Java e o mundo aparecerá na sua lista de jogos para um jogador.

## 📱 Para Minecraft Bedrock (Console, Celular, Windows 10/11)
Localize o arquivo .mcworld no seu gerenciador de arquivos.
Clique duas vezes (ou toque) no arquivo.
O Minecraft Bedrock será iniciado automaticamente e começará o processo de importação.
Assim que terminar, o mundo estará disponível no topo da sua lista de Mundos.
