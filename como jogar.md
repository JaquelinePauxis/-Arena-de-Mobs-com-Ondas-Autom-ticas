## ⚔ Projeto MEE 2 — Arena de Mobs com Ondas Automáticas
✅ Comandos principais (professor/aluno)

💬 arena → constrói a arena e define o centro

💬 start → inicia o jogo e as ondas automáticas

💬 stop → para o jogo imediatamente

💬 status → mostra onda, pontos, vidas e “mobs vivos”

💬 limpar → limpa a arena (remove mobs) — útil para controle de sala

## Sequência recomendada de uso em aula

1. Abrir chat ( Tecla T )

2. usar comando: arena

3. usar comando: start

Durante o jogo: 
4. usar comando: status

Se sair do controle: 
5. usar comando: limpar

Para finalizar: 
6. usar comando: stop
